## Structure

```
ferrari-luce/
├── index.html        # Main page
├── assets/
│   └── ferrari_luce.png   # Hero image (add yours here)
└── README.md
```

## Setup

1. Clone the repo
2. Place your car image at `assets/ferrari_luce.png`
3. Open `index.html` in a browser — or deploy via GitHub Pages

## GitHub Pages

In your repo settings → Pages → set source to `main` branch / `root`.  
Your site will be live at `https://<your-username>.github.io/ferrari-luce/`
